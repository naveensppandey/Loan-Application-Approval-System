# 🏦 Loan Application & Approval System

> **Azure DevOps Manual Testing Project | Test Case Design**

## 📌 Project Overview

This project demonstrates the complete manual testing documentation for a **Loan Application & Approval System** using **Behavior-Driven Development (BDD)** in Azure DevOps.

The objective is to validate the **Loan Disbursement Notification** module by creating well-structured BDD test cases, defining acceptance criteria, and documenting the workflow using Azure DevOps.
